#<Cryo Element>

    #<Description>

        By Aaron0000, Kazy, Mike, FromDarkHell, Shadowevil, and The Campin' Dutchman.

        Turns Slag into a facsimile of the Pre-Sequel Cryo element.

        Features;

        +125% movement slowdown.

        +250% melee damage.

        +200% explosive damage.

        +200% extra critical hit damage.

        White glows on items that use cryo.

        Slows enemies down when frozen. (Or stops movement completely, if you use the alternative version.)

        Reduces fire rate by 50% when frozen. (There is also an alternate version that removes the fire rate penalty.)

        Damage over time.

        Recolored damage numbers.

        Adjusted prefixes/names.

        Reskins for the Grog Nozzle and Slagga (which was renamed to Freeza). Off by default.

        It also includes Pre-Sequel weapon replacers for the following (all of which are off by default);

        Chulainn - Frostfire

        Pyrophobia - Cryophobia

        Fastball - Snowball (but still able to spawn in the other elements)

        Gub - Zim

        Lascaux - Fridgia

        Credits;

        Aaron for the basic functions/attributes, the Pre-Sequel item ports, the reskins, some gfx, and prefix/title name changes.

        Kazy for making some text changes, the damage number color, and part of the item glows.

        Mike for a lot of the gfx visual stuff and some other visual bits.

        FromDarkHell for the "freezing" of movement speed option.

        Shadowevil for making Cryo DoTs appear on the item card.

        The Campin' Dutchman for advice and coaching.

        Notes;

        If you get stuck in a pool of slag while using the version that stops movement, just spam jump and you should eventually get out.

        There is an option to switch the complete stopping of movement with a heavily reduced movement speed instead.

        This file contains a hotfix to change the name of "Slagged" enemies to icy versions, as well as change the color of Scorn to white. Be sure to either merge this file with your main file containing hotfixes, or disable the hotfix (it is disabled by default).

        There are GFX and visuals that have yet to be adjusted, due to difficulties in actually doing so (if even possible). These will (hopefully) be resolved in a future update.

    #</Description>
