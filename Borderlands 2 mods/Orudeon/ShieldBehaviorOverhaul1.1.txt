#<ShieldBehaviorOverhaul>

    Buffs most shield types (sometimes majorly) to improve overall usability and help specialized or common shields compete with generalized and unique-or-better ones. This mod is designed to be used with Shield Stat Tweaks and requires the Unofficial Community Patch v3.1 or better for Amp shield functionality.

    Coded by: Orudeon, with major help from LightChaosman (coded the Amp Shield behavior), AngrierPat, zububu, et al. It is highly recommended you use LightChaosman's Filter Tool to install this.

    #<Amp Shields>

        Redesigns common Amp Shields to constantly regenerate, apply Amp and drain every shot insofar as there's enough shield to drain, and triples common Amp shield Amp damage to help them better compete with The Bee. Please thank LightChaosman for coding the Amp/Drain on every shot behavior.

        If you'd prefer regular delay values, you can optionally enable that instead. This code defaults to No Delay if not configured.

        Required Code:

        set GD_Shields.Skills.Impact_Shield_Skill:AttributeExpressionEvaluator_20 Expression (AttributeOperand1=ResourcePoolAttributeDefinition'D_Attributes.ShieldResourcePool.ShieldCurrentValue',ComparisonOperator=OPERATOR_GreaterThanOrEqual,Operand2Usage=OPERAND_PreferAttribute,AttributeOperand2=AttributeDefinition'D_Attributes.Shield.ImpactShield_ConsumptionOnEmpower',ConstantOperand2=0.000000)
        #<hotfix><key>"SparkLevelPatchEntry-ampshieldContinuousDrain1"</key><value>",GD_Shields.Skills.Impact_Shield_Skill:BehaviorProviderDefinition_0,BehaviorSequences[0].ConsolidatedLinkedVariables,,(0,0,1,0,2,1,0,1,1,0,0)"</value><on>

        set GD_Shields.A_Item.Shield_Impact:AttributePresentationDefinition_0 Description Weapon shots deal bonus [skill]Amp Damage[-skill].

        set GD_Shields.A_Item.Shield_Impact AttributeSlotUpgrades ((SlotName="Capacity",GradeIncrease=0,bActivateSlot=True),(SlotName="RechargeRate",GradeIncrease=0,bActivateSlot=True),(SlotName="RechargeDelay",GradeIncrease=0,bActivateSlot=True),(SlotName="Special01",GradeIncrease=0,bActivateSlot=True),(SlotName="Special02",GradeIncrease=0,bActivateSlot=True),(SlotName="Special03",GradeIncrease=0,bActivateSlot=True))
		
		NOTE: YOU MUST INSTALL THIS WITH FILTER TOOL OR MANUALLY MERGE THE HOTFIXES AT THE END OF THE MOD OR AMP DRAIN BEHAVIOR WILL NOT FUNCTION PROPERLY.

		1.1a: Added dependency code from UCP 3.1.
