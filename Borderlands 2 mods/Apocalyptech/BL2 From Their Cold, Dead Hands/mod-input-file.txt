    # Enemies will always drop the specific gun/shield that they were using, on death, and
    # the gear quality for enemies has been much improved.  You will be fighting many
    # enemies which have blue and higher gear (including legendaries).  Badasses should
    # have better-quality gear than regular enemies, in general.  Creatures like Stalkers
    # and Skeletons, and devices like turrets, will NOT drop a shield, even if they spawn
    # with one.  Shielded loaders will drop shields, though.
    #
    # This mod requires that the UCP option "Skinpool Fixes (Don't uncheck this)" is
    # enabled!  If you want to run this without UCP, make sure to at least copy that
    # category over.

    #<Disable World Weapon/Shield Drops>

        # This section disables world drops for weapons and shields, so those items will
        # only drop from what the enemies are carrying.  Rather than deleting/disabling
        # the items entirely, we're changing them to draw from an ammo/money pool so that
        # the rest of the probabilities in the pools aren't affected.
        #
        # Note that this doesn't *completely* get rid of world weapon/shield drops, but it
        # should get rid of the vast majority of them.
